## Managing Direct Debit Instructions

A Direct Debit Instruction (DDI) is an authorisation to collect future payments from a nominated account.
A DDI can be submitted to ClearBank速 by Service Users electronically via the Bacs Automated Direct Debit Instruction Service (AUDDIS) or sent as a paper instruction to your address to be entered via the ClearBank速 API after you have validated it.
A DDI can be set up against real and virtual accounts held with ClearBank速.

If you are transferring an existing sort code to ClearBank速 that already has DDIs set up, you will need to use the API to inform us of these.
